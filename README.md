# Basic CRUD Operation

## Technology

- REST framework (FastAPI)

## Prerequisite

- Python3

## Installation

Windows user

```
python -m venv fastapi_dev
fastapi_dev\Scripts\activate.bat
pip install -r requirements.txt
```

Mac use

```
python -m venv fastapi_dev
source fastapi_dev/bin/activate
pip install -r requirements.txt
```

## Start

```
python main.py
```
