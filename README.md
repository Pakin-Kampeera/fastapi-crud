# Basic CRUD Operation

## Technology

- REST framework (FastAPI)

## Prerequisite

- Python3

## Installation

Windows user

```
python3 -m venv fastapi_dev
fastapi_dev\Scripts\activate.bat
pip install -r requirements.txt
```

Mac use

```
python3 -m venv fastapi_dev
source fastapi_dev/bin/activate
pip install -r requirements.txt
```

## Start

```
python main.py
```
http://127.0.0.1:8000/docs
